## Hulpvragen 

### Help in mijn transcriptie komen vreemde herhalingen voor
Indien je in je transcriptie vreemde herhalingen tegenkomt zoals "TV GELDERLAND" dan betekend dit dat de audio kwaliteit niet voldoende was om een zin te maken. 
Dit komt ook voor als de opname tijdens pauzes aan blijft staan en er dus geen gesprek was om op te nemen. <br/>  
🫰**Tip**: Zet je opname op pauze wanneer er een pauze in je sessie is. 

### Help hoe check ik mijn audio kwaliteit? 
De audio opname wordt in het dashboard opgeslaan tot 30 dagen nadat hij is toegevoegd. Binnen deze tijd kan je bovenaan het transcript de audio speler aanzetten. Dit zorgt dat je de audio van de tekstfragmenten opnieuw kan luisteren. 

Navigeer naar stukken in het het transcript waar vreemde herhalingen voorkomen. Of stukken die heel erg kort zijn, bijvoorbeeld maar 1 zin. 
- Luister naar deze stukken. 
- Als je kan verstaan wat er gezegt wordt maar dit komt niet terug in jouw transcriptie, kijk dan naar de volgende hulpvraag. 
- Versta je ook niet wat er gezegd is? Dan is de audiokwaliteit jammer genoeg te laag om een goede transcriptie te bekomen. 
![](/DocsDashboard_Audiospeler.png) <br/>  

### Help mijn transcriptie kwaliteit is slecht maar mijn audio is wel oke
Jammer dat dit is voorgegevallen. We doen ons best om dit te vermijden en de transcriptie direct juist te laten verlopen. Maar gelukkig, als de audio goed is kunnen we de transcriptie opnieuw laten uitvoeren. 
 - Klik op het icoon van de rondraaiende pijltjes om de hertranscriptie te starten. 
![](/Docs_Hertranscriptie.png) <br/>  



### Help Ik zie een gesprek niet in het dashboard verschijnen
1. Heb je gekeken of het portaal goed aan het opnemen is ? Geen zwart scherm en goede wifi verbinding?
2. Indien dit zo is dan kan het zijn dat er een kleine server congestie is. <br/> 
🧪 Je kan dit testen door de QR code opnieuw te scannen en een nieuw gesprek te starten. <br/>
 &nbsp;&nbsp;&nbsp;&nbsp; Verschijnt deze wel in het dashboard? <br/> 
 &nbsp;&nbsp;&nbsp;&nbsp; Ga dan door met dit gesprek en sluit het andere af. <br/> 
 &nbsp;&nbsp;&nbsp;&nbsp; Verschijnt deze ook niet? <br/> 
 &nbsp;&nbsp;&nbsp;&nbsp; Dan heeft de server het even druk. In dit geval moet je even wachten. <br/> 
&nbsp;&nbsp;&nbsp;&nbsp; Alles gaat wel gewoon goed achter de schermen.

### Help ik heb geen internet 
Zit je op een locatie zonder internet en wil je toch Dembrane later gebruiken? Neem dan het gesprek op met je telefoon. Deze kan je later uploaden via het dashboard. Dit door je door op het + icoontje te klikken naast 'Gesprekken'. Dit opent je file explorer waar je een bestand kan selecteren. Als je in de rechteronderhoek van je scherm een melding ziet dat de upload succesvol was, maar je ziet hem nog niet verschijnen. Ververs je browser dan even en dan verschijnt de opname onder gesprekken. 

🪲 Indien dit niet zo is, maak dan een foutmelding zoals hierboven staat beschreven.
