## Stap 1: Je dashboard instellen

1. Log in op je Dembrane account
2. Klik op het **Instellingen** icoon rechtsboven in je dashboard om de taal van je dashboard te wijzigen of een probleem te melden. Als je op enig moment wilt uitloggen, kan dat hier ook.

![](/img13.png)

## Stap 2: Project Aanmaken

1. Als je een nieuw project wilt maken, klik dan op het Home icoon en klik vervolgens op "Aanmaken".

![](/img1.png)

### Project Instellingen

Nu kom je in je hoofddashboard terecht. Zorg eerst dat je in het tabblad **"Project Instellingen"** bent. Hier geef je je project een naam en vermeld je wat over de context van jouw project.
1. **Naam**: Geef je project een naam in plaats van **"Nieuw Project"**. Kies iets beschrijvends en herkenbaars. Je kunt dit later nog wijzigen
2. **Context**: Voeg details toe over het doel van je project (intern gebruik). Dit kan bijvoorbeeld zijn wat het doel van het project is.

![](/img2.png) 

Wat je verder nog in de project instellingen kan doen is alle transcripties downloaden en het project verwijderen wanneer deze niet meer nodig is.





## Stap 3: Je portaal instellen

Het deelnemersportal is je toegangspoort tot het verzamelen van gesprekken op grote schaal. Hier configureer je wat je deelnemers gaan zien bij het scannen van de QR code van het project en in welke taal de transcripten worden opgeslagen.

Voor de volgende aanpassingen te doen, zorg eerst dat je in het tabblad **"Portal Editor"** bent. Deze staat langs de projectinstellingen.

![](/img14.png)


#### Taalinstellingen
1. **Taal**: Dit is de taal van het deelnemersportaal en de transcriptie. <br/>
    Heb je deze in engels staan en je gespreken worden gevoerd in Nederlands, dan gaat je opgenomen audio naar engels vertaald worden en zo worden opgeslagen. <br/>
    ü´∞Zorg dus altijd dat de taalinstelling van het portaal dezelfde is als de taal van jouw dialoogsessie om correcte transcriptie te verzekeren. <br/>
    Verwacht je meerdere talen? Maak hier dan verschillende projecten voor.  <br/>
    ‚ö†Ô∏è Als je de taal van het project verandert, dan verandert ook de QR code.
2. **Vraag om Naam**: Door dit vakje aan te vinken, moeten deelnemers hun naam opgeven wanneer ze een gesprek starten. 

![](/img5.png)

#### Tutorial Opties
Bij het scannen van de QR code krijgen je deelnemers een tutorial te zien. Door te klikken op het knopje "Show preview" kan je zien hoe dit er voor de deelnemer eruit gaat zien.

![](/PortalEditor_ShowPreview.png)

Je hebt de keuze uit drie opties:
1. **Geen Tutorial**
   - Dit bevat alleen de privacy-informatie
2. **Basis**
   - Dit toont de privacy-informatie en een introductie tot Dembrane.
3. **Geavanceerd**
    - Dit toont de privacy-informatie, een introductie tot Dembrane en tips en trics voor de deelnemer om zo een goed mogelijke opname te maken.

#### Trefwoorden
Hier kan je trefwoorden aanmaken die deelnemers kunnen kiezen bij het starten van een opname. Achteraf kan je ook nog trefwoorden toevoegen aan het gesprek. Dit is handig om gesprekken makkelijk terug te vinden later.
![](/PortalEditor_Trefwoorden.png)


#### Pagina-instellingen
1. **Pagina titel**: Deze titel wordt getoond aan deelnemers wanneer ze een gesprek starten
2. **Pagina inhoud:** Deze pagina wordt getoond aan deelnemers wanneer ze een gesprek starten na het succesvol afronden van de tutorial
3. **Bedank pagina inhoud**: Deze pagina wordt getoond nadat de deelnemer het gesprek heeft afgerond

![](/img6.png) 

#### Transcriptie-instellingen
**Specifieke context:** Hier kun je belangrijke termen, afkortingen of namen van sprekers toevoegen om zeker te zijn dat deze juist in de transcriptie voor komen. Merk je dat in de analyse van je gevegens consquent hetzelfde foutgeschreven staat? Dan is dit de plek om het woord toe te voegen zodat het juist in de analyse verschijnt.
