# Kernconcepten

De kernconcepten van ECHO zijn:

- **Portaal** - Het portaal is waar deelnemers kunnen deelnemen aan het gesprek. De QR-code die gescand wordt, leidt een deelnemer naar het portaal. 
- **Dashboard** - Dit is de plek waar de gesprekken binnenkomen en je de analyse kan uitvoeren.
- **Stel vraag** - De Stel vraag interface is waar je kunt chatten met een selectie van gesprekken.
- **Bibliotheek** - De bibliotheek is de plek waar je inzichten kunt creëren uit AL je gesprekken.<br/>
 &nbsp;&nbsp;&nbsp;&nbsp; - **Views** - Views zijn perspectieven waarmee je alle gesprekken in een project kunt analyseren.  <br/>
&nbsp;&nbsp;&nbsp;&nbsp; - **Insights** - Insights zijn clusters van citaten die ECHO heeft gecreëerd uit alle gesprekken in een project.<br/>
- **Rapport** - Dit is het rapport dat wordt gegenereerd vanuit al de gesprekken van het project.
